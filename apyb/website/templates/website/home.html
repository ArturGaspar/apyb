A APyB

A Associação Python Brasil (APyB) é uma associação sem fins lucrativos formada em abril de 2007 com objetivo de apoiar as comunidades relacionadas à linguagem Python e suas tecnologias derivadas no Brasil.

Ela é composta por uma diretoria, um conselho deliberativo, um conselho fiscal, suplentes do conselho e o corpo de associados.

Atualmente a APyB tem como seu principal objetivo atuar como um canal facilitador para a comunidade realizar o evento nacional da linguagem Python (Python Brasil), bem como para os eventos regionais.
Intermediando o processo de captação de recursos, documentando processos, conectando pessoas, fornecendo infra estrutura, dentre outros.
